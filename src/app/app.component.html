TASKERY

This project is a web-based application designed to help users manage and control their project tasks.
It provides a user-friendly interface in English and Ukrainian to plan, execute, and monitor tasks and projects. Key features include user authentication, creating and describing projects using artificial intelligence, tracking project and task statuses, generating reports, and viewing the calendar. The project aims to improve the user experience, simplify workflow organization, improve communication and team productivity, and ensure the quality of tasks and deadlines.
