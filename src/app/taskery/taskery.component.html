Taskery

A user-friendly app designed for efficient project task management. With a seamless interface available in English and Ukrainian, users can easily plan, execute, and monitor their tasks and projects. Stay organized, collaborate with your team, and make data-driven decisions to optimize workflow.

On the main page, the role differentiation starts
Team Members have an additional link to the daily task list, divided into three segments: overdue tasks, tasks for today, and upcoming tasks (one or two days before the deadline).

Each user has access to only those projects to which they are assigned. Project Manager have an additional button to create new project.

Calendar with all tasks color-coded by project

It displays a detailed report of the specific project, which includes the budget, project start date, today's date, number of tasks, number of overdue tasks, project status, the pie chart of task status, and the chart of the employee task status.

The developed web application fulfills its purpose, improving communication between team members, ensuring efficient resource allocation, increasing productivity, facilitating quality control as well as meeting deadlines.
